1. HTML

Стандартизированный язык разметки документов для просмотра веб-страниц в браузере.

2. CSS

(Cascading Style Sheets) - это код, который вы используете для стилизации вашей веб-страницы.

3. Figma

2. SAAS

SAAS (Software as a Service) и SCSS (Sassy CSS) — это два разных термина, не связанных между собой.

3. SCSS

SCSS расширяет возможности стандартного CSS, добавляя переменные, вложенность, миксины, функции и другие дополнительные функции.

1. SCSS

SCSS расширяет возможности стандартного CSS, добавляя переменные, вложенность, миксины, функции и другие дополнительные функции.

2. BOOTSTRAP

Bootstrap — это фреймворк с открытым исходным кодом для создания веб-сайтов и веб-приложений.

3. GIT

Git и очень популярна среди разработчиков, работающих над открытыми и закрытыми проектами.

1. GITHUB

GitHub — это онлайн-платформа для хранения, совместной работы и управления версиями кода.

2. JS

JavaScript — это высокоуровневый, интерпретируемый язык программирования, который широко используется для создания динамического и интерактивного контента на веб-страницах.

3. LIBRARY

JavaScript библиотеки — это наборы готовых функций и классов, которые помогают разработчикам ускорить процесс разработки, предоставляя уже написанный и протестированный код для выполнения распространенных задач. Использование библиотек позволяет избежать повторного изобретения колеса и сосредоточиться на реализации логики приложения.
